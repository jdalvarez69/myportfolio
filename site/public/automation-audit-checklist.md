# Automation Audit Checklist

Use this checklist to identify quick automation wins for small businesses.

- Capture current manual tasks and frequency (daily/weekly/monthly)
- Identify data sources (Google Sheets, emails, forms, CRM)
- Prioritize tasks by time saved and error reduction
- Check for existing integrations (Zapier, Make, native CRM connectors)
- Define required outputs (PDF invoices, email notifications, dashboard KPIs)
- Determine authentication and data access needs
- Estimate scope and effort (hours) and propose Starter Pack if small

Bring this checklist to your free 20-minute Automation Audit.
